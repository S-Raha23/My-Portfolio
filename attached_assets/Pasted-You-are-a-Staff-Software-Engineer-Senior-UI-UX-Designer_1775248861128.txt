🎯 GOAL:
Create a highly distinctive, modern portfolio for:

* AI/ML enthusiast
* Strong DSA background (C++)
* Aspiring Software Engineer

* FAANG-level recruiters
* senior engineers
* hackathon judges

---

⚠️ STRICT REQUIREMENTS (CRITICAL):

* NO generic hero text (“passionate developer” etc.)

🔥 DESIGN REQUIREMENTS:

* Modern, minimal, but distinctive (NOT template)

* glassmorphism OR minimal gradients

⚙️ TECH STACK (FREE ONLY):

* Next.js (App Router)

* production-ready
